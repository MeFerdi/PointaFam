### PointaFam

PointaFam is a web-based application aiming to connect farmers with urban retailers, providing a platform for direct trade of fresh produce.

# Features
Connect Farmers and Urban Retailers: PointaFam provides a platform for farmers to list their fresh produce and urban retailers to browse and purchase directly from them.

Efficient Trade: By eliminating intermediaries, PointaFam facilitates efficient and transparent trade between farmers and urban retailers, resulting in fair prices for farmers and fresh produce for retailers.

User Authentication: Secure user authentication system for farmers and urban retailers to create accounts, log in, and manage their profiles.

Product Listings: Farmers can list their produce for sale, including details such as product name, description, quantity, pricing, and photos.

Order Management: Urban retailers can browse available products, place orders, and manage their purchases through the platform.

# Technologies Used
Astro: Modern web framework for building static websites with dynamic components.
Tailwind CSS: Utility-first CSS framework for styling the application.
htmx: Lightweight JavaScript library for adding dynamic behavior to HTML elements.
Alpine.js: Minimal JavaScript framework for interactivity and managing client-side state.
Pocketbase: Serverless database service for storing and retrieving data.

# Usage
Once the application is running, farmers can sign up, list their produce, and manage their listings. Urban retailers can sign up, browse available products, place orders, and manage their purchases.
